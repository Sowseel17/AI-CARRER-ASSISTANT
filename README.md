# 🚀 AI Career Assistant

An AI-powered web application that helps job seekers optimize their resumes by analyzing them against a Job Description (JD), identifying skill gaps, suggesting resume improvements, tracking resume progress, and ranking multiple resumes based on their suitability for a job.

# 📌 What does this application do?

AI Career Assistant helps candidates optimize their resumes for specific job roles using Artificial Intelligence.

It evaluates resumes against a Job Description, provides ATS-based recommendations, tracks resume improvements, and can compare multiple resumes to identify the best candidate for a particular role.

---

# ✨ Features

## 📄 Resume ATS Analyzer

- Upload Resume PDF
- Paste Job Description
- ATS Compatibility Score
- Professional Resume Summary
- Matching Skills
- Missing Skills
- Resume Strengths
- Areas for Improvement
- Resume Optimization Recommendations
- Resume Enhancement Suggestions
- ATS Keyword Recommendations
- Personalized Learning Roadmap

---

## 📈 Resume Progress Tracker

- Upload Updated Resume
- Compare with Previously Analyzed Resume
- ATS Score Improvement
- Newly Added Skills
- Removed Skills
- Improved Resume Sections
- Remaining Improvement Areas
- AI Generated Overall Feedback

---

## 👥 Multiple Resume Comparison

- Upload multiple resumes for the same Job Description
- Analyze every resume individually
- Compare ATS scores
- Rank resumes based on their suitability
- Identify the best matching candidate
- Provide AI-generated reasoning for the ranking
